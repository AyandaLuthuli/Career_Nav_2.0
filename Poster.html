Solution Overview

Stack: HTML/CSS/JS (frontend), Supabase (DB/auth), Google Gemini (AI & NLP). The chatbot collects user profiles (name, education, skills, interests), generates 3 personalised career suggestions, and stores results for future sessions.

Architecture

Frontend → Supabase (users & recommendations) → Gemini API for reasoning. Hybrid model: local curated data + generative AI for up-to-date suggestions.
